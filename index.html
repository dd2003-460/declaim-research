📊 Paper: "Detecting Scientific Paradigm Shifts"

Detecting Scientific Paradigm Shifts via Automated Topic Changepoint Analysis: Lessons from Rigorous Validation

Target venues: ACL FinNLP Workshop → arXiv → EMNLP / AAAI Workshop

Methodology

  1. Data: ~10K arXiv abstracts (quantum computing, 2020–2025)
  2. Model: LDA (15 topics) → topic proportion time series
  3. Detection: Bayesian changepoint detection (posterior > 0.95)
  4. Validation: Event study method (4 windows) + 100 randomized control dates
  5. Correction: Bonferroni multiple testing adjustment + power analysis
